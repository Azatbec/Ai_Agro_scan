## 2. Введение

**A) Актуальность темы:**
В современном сельском хозяйстве болезни растений являются одной из главных причин потери урожая (до 40% ежегодно). Своевременное обнаружение патогенов на ранних стадиях позволяет минимизировать использование химикатов и сохранить продукцию. Автоматизация этого процесса с помощью компьютерного зрения (CV) и методов цифровой обработки изображений (DIP) является критически важной задачей для повышения эффективности агропромышленного комплекса.

**B) Цель курсовой работы:**
Целью работы является разработка программного комплекса **AgroScan AI** для автоматической диагностики заболеваний растений по фотографиям листьев с использованием гибридного подхода: классических методов фильтрации и сегментации (DIP) в сочетании с глубоким обучением (Deep Learning).

**C) Задачи работы:**
1. Изучить теоретические основы цифровой обработки изображений и архитектуры современных нейронных сетей.
2. Реализовать алгоритмы предварительной обработки (CLAHE, Canny) и сегментации (HSV thresholding).
3. Обучить и интегрировать модель глубокого обучения (EfficientNet) для классификации 38+ типов состояний растений.
4. Разработать интерактивный дашборд для визуализации результатов DIP-анализа и AI-прогноза.
5. Провести оценку точности системы на тестовом наборе данных.

**D) Объект исследования:**
Процесс автоматизированного анализа цифровых изображений вегетативных органов растений.

**E) Предмет исследования:**
Алгоритмы предварительной обработки, сегментации и глубокого обучения для идентификации фитопатологий.

**F) Используемые инструменты и библиотеки:**
*   **Язык:** Python 3.10+
*   **DIP:** OpenCV, NumPy, Pillow
*   **AI/ML:** PyTorch, Torchvision
*   **Интерфейс:** Streamlit, Plotly (для графиков)
*   **Среда разработки:** VS Code / PyCharm / Jupyter Notebook

## 3. Постановка задачи
Необходимо разработать программное приложение, которое:
1. Позволяет пользователю загружать изображение пораженного листа растения.
2. Выполняет предварительную обработку для улучшения видимости признаков болезни.
3. Проводит DIP-анализ (выделение границ, сегментация по цвету, построение гистограмм).
4. Использует предобученную нейросеть для определения конкретного заболевания и вероятности диагноза.
5. Предоставляет агрономические рекомендации по лечению.

## 4. Теоретическая часть

**A) Цифровое изображение:** представляет собой двумерную матрицу пикселей.
**B) Пиксель:** наименьший элемент изображения, содержащий информацию о цвете и яркости.
**C) Цветовые модели:**
*   **RGB:** (Red, Green, Blue) — базовая модель для отображения на экранах.
*   **HSV:** (Hue, Saturation, Value) — удобна для сегментации, так как разделяет информацию о цвете (цветовой тон) и яркости.
*   **Grayscale:** оттенки серого, используются для упрощения вычислений при поиске границ.

**D) Предварительная обработка:**
*   **CLAHE (Contrast Limited Adaptive Histogram Equalization):** Метод адаптивного выравнивания гистограммы для улучшения локального контраста без усиления шума. Позволяет лучше видеть мелкие пятна болезней на листе.
*   **Удаление шума:** Медианный фильтр для устранения импульсных помех.

**E) Выделение границ:**
*   **Детектор Кэнни (Canny Edge Detector):** Многоступенчатый алгоритм для обнаружения широкого спектра границ. Используется для анализа текстуры листа и формы пятен.

**F) Сегментация:**
*   **Пороговая сегментация в пространстве HSV:** Позволяет отделить зеленый лист от коричневой почвы или серого фона путем задания диапазонов цветового тона.

**G) Глубокое обучение:**
*   **EfficientNet:** Современная архитектура сверточных нейронных сетей, оптимизированная по количеству параметров и точности.

## 5. Описание используемых методов

| Метод | Обоснование выбора |
| :--- | :--- |
| **CLAHE** | Выбран для компенсации неравномерного освещения в полевых условиях. Усиливает контрастность между здоровой тканью и очагом инфекции. |
| **Canny** | Позволяет четко локализовать границы поражения и структуру жилок листа. |
| **HSV Thresholding** | Идеален для агро-задач, так как позволяет легко выделить хлорофилл-содержащие ткани (зеленый спектр) и отсечь фон. |
| **EfficientNet-B0** | Обеспечивает высокую скорость инференса на мобильных устройствах при сохранении высокой точности классификации. |

## 6. Описание программной реализации

**A) Язык и библиотеки:** Приложение написано на Python. Для UI выбран Streamlit за его способность быстро создавать интерактивные DIP-инструменты.
**B) Структура программы:**
*   `core/pipeline.py`: Ядро системы, управляющее загрузкой моделей и инференсом.
*   `scripts/app.py`: Главный файл интерфейса с двумя вкладками (AI Диагностика и DIP Анализ).
*   `models/`: Содержит веса обученных нейросетей (.pth).
**C) Основные функции:**
*   `apply_dip()`: Выполняет CLAHE, Canny и HSV-маскирование.
*   `run_inference()`: Прогоняет изображение через нейросеть с применением фильтра культуры.
**D) Входные данные:** Файлы JPG/PNG размером до 200MB.
**E) Выходные данные:** Название болезни, процент уверенности, DIP-визуализации, рекомендации.
**F) Репозиторий:** Проект размещен на GitHub (AgroScan-AI).

## 8. Анализ результатов

В ходе тестирования были получены следующие метрики (согласно `full_report.txt`):
*   **Общая точность (Accuracy): 89%**.
*   **Средневзвешенная F1-мера: 0.87**.
*   **Лучшие результаты:** Томат (Bacterial spot, Late blight) — точность > 99%.
*   **Сложные случаи:** Некоторые виды кукурузы (Common Rust) показали точность около 66-80% из-за схожести текстур на ранних стадиях.

DIP-анализ показал, что применение CLAHE перед подачей в нейросеть повышает уверенность модели (Confidence) в среднем на 4-6% для изображений с плохим освещением.
