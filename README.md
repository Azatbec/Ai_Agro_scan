## 📝 Описание проекта
**AgroScan AI** — это комплексная программная платформа, предназначенная для автоматизации диагностики заболеваний сельскохозяйственных культур. Система объединяет современные методы глубокого обучения (Deep Learning) и классические алгоритмы цифровой обработки изображений (DIP), предоставляя пользователю не только диагноз, но и глубокую техническую аналитику снимка.

---

## 🌟 Ключевые модули и функционал

### 1. 🧠 Интеллектуальная диагностика (AI Core)
* **Архитектура**: Нейросеть **EfficientNet-B0**, предобученная на ImageNet и дообученная (Fine-tuning) на специализированном датасете болезней растений.
* **Область распознавания**: 68 уникальных классов (культуры: томат, картофель, кукуруза, рис, яблоня, виноград, перец и др.).
* **Скорость**: Среднее время инференса на CPU — 120-150 мс.
* **Локализация**: Автоматический перевод названий болезней и рекомендаций на **RU**, **KZ** и **EN**.

### 2. 🔬 Цифровая обработка изображений (DIP)
Специальный модуль для технического анализа, демонстрирующий применение классических алгоритмов:
* **CLAHE (Contrast Limited Adaptive Histogram Equalization)**: Алгоритм адаптивного выравнивания гистограммы для компенсации неравномерного освещения и выделения деталей болезни.
* **Canny Edge Detection**: Детектор границ для анализа текстурных изменений поверхности листа.
* **HSV Segmentation**: Сегментация в цветовом пространстве HSV для выделения полезной площади листа и фильтрации фона (почвы, сорняков).
* **Гистограммный анализ**: Визуализация распределения интенсивностей пикселей для оценки качества снимка.

### 3. 📈 Аналитика и Метрики (Model Evaluation)
Система включает автономный модуль `model_analytics.py` для оценки качества обучения:
* **Confusion Matrix**: Детальная тепловая карта ошибок модели.
* **Precision/Recall/F1-Score**: Расчет научных метрик точности для каждого из 68 классов.
* **Статистика площадей**: Вычисление процентного соотношения площади листа к фону на основе сегментации.

## 🛠 Технологический стек
* **Frontend**: Streamlit с глубокой кастомизацией через CSS (Glassmorphism design).
* **Backend**: PyTorch (модели), OpenCV (DIP-алгоритмы).
* **Визуализация данных**: Plotly (интерактивные графики), Seaborn и Matplotlib (статические отчеты).
* **Обработка данных**: Pandas, NumPy, Scikit-learn.

## 🚀 Инструкция по развертыванию

### 1. Подготовка окружения
Рекомендуется использовать Python 3.10. Создайте виртуальное окружение:
```bash
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
```

### 2. Запуск системы
Для запуска пользовательского интерфейса:
```bash
.\run_dashboard.bat
```

### 3. Генерация технических отчетов
Чтобы получить графики точности и матрицу ошибок:
```bash
.\.venv\Scripts\python.exe scripts/model_analytics.py
```
*Результаты будут сохранены в папку `reports/`.*
